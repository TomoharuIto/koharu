SHICHIMI
===

## Overview
SHICHIMI(シチミ)は、長野県内の天気予報を提供する為のTwitter botです。  
毎朝６時に長野県内の天気予報を吐きます。  
またダイレクトメッセージから各地域の天気へのリンクを取得できます。
